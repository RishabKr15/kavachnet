# KavachNet

KavachNet is an AI-based VPN & Proxy Detector tool.

## Installation

```bash
pip install kavachnet
```

## Usage

To launch the web interface:

```bash
kavachnet-web
```

## Features
- Detect VPN and Proxy IP addresses
- Web-based interface using Streamlit
